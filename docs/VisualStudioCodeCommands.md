## Command to list Visual Studio Code Extensions

```powershell
code --list-extensions
```

## Command to list Visual Studio Code Extensions with versions

```powershell
code --list-extensions --show-versions
```

## Command to install extension

```powershell
code --install-extension github.copilot-chat
```

## Command to install Specific Version of extension

```powershell
code --install-extension continue.continue@1.0.5
```

## Command to uninstall an extension

```powershell
code --uninstall-extension <PublisherName>.<ExtensionName>
```

## Command to disable all extensions globally

```powershell
code --disable-extensions
```
